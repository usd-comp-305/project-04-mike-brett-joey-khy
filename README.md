# USD Casino

A terminal based Java casino simulator featuring five classic games: Slots, Craps, Blackjack, Baccarat, and Roulette. 
Players start with a $1,000 balance and interact through the terminal to place bets and make decisions. 
The project demonstrates core OOP principles including interface usage, the Factory pattern, and the State pattern (used in Craps).

## Prerequisites

- No external libraries or build tools required

## Installation

- Nothing extra needed to install. The user is able to simply run the program in their terminal.
